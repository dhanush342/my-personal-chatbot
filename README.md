# My Personal Chatbot

An AI-powered chatbot built with Python that provides intelligent conversational responses using Google's Gemini AI model.

## Features
- Natural language processing using Gemini AI
- Interactive command-line interface
- Environment-based configuration for secure API key management
- Conversation history tracking

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your `GEMINI_API_KEY`
4. Run: `python bot.py`

## Requirements
- Python 3.7+
- google-generativeai
- python-dotenv
